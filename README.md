# LexTV 词境 — Android TV 新闻英语单词软件

深墨蓝印刷词典风格,专为 85 寸大屏 + 遥控器设计。FSRS-4.5 间隔重复调度。

## 内置词库(可离线)
- 📰 新闻核心 ~1200 词(政治选举/冲突军事/法律司法/灾害气候健康/社会经济/报道动词与媒体)
- 🤖 科技媒体 ~300 词(AI软件/芯片硬件通信)
- 共 1392 个去重单词,每词含音标、词性、中文释义、新闻语境例句

## 五种玩法
学新词(卡片翻转) / 智能复习(FSRS到期调度) / 闪电测验(限时四选一+连击) / 听音辨义(只听发音) / 例句填空(语境激活)

## 遥控器操作
- 主页:方向键选择,OK 进入,返回键退出应用
- 学习卡:OK 翻面/重听,◀不认识 ▼模糊 ▶认识(未翻面按▶=秒过认识的词)
- 测验:方向键选,OK 确认,菜单/播放键重听
- 返回键:任何界面回主页

## 打包安装(与坦克大战流程相同)
1. 把本项目上传到 GitHub 仓库(整个文件夹内容)
2. Actions 会自动运行 "Build APK",完成后下载 LexTV-apk 工件
3. 解压得到 LexTV.apk,拷入 U 盘,电视上用文件管理器安装

## 之后添加词书(两个口子)
- 口子A(推荐):把新词书 .json 放进 `app/src/main/assets/decks/`,在 manifest.json 登记,重新打包
- 口子B(免打包):电视上把 .json 复制到 `/sdcard/Android/data/com.lextv.app/files/decks/`,重启应用自动加载
- 词条格式:`[["单词","音标","词性 中文释义","英文例句"], ...]`

## 说明
- 发音使用电视系统 TTS;若国内电视缺英文 TTS 引擎,设置页会提示"发音不可用",纯视觉学习不受影响
- 进度存本机,设置页可重置
