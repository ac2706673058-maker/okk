# 坦克大战 TV 版 — 编译 APK 说明

## 一、最快路径(Android Studio,约10分钟)

1. 电脑安装 [Android Studio](https://developer.android.com/studio)(国内可用中文官网镜像)
2. 打开 Android Studio → **Open** → 选择本工程文件夹 `BattleCityTV`
3. 等待 Gradle 同步完成(首次会自动下载依赖,需要网络)
4. 菜单 **Build → Build App Bundle(s) / APK(s) → Build APK(s)**
5. 完成后点右下角提示的 **locate**,APK 在:
   `app/build/outputs/apk/release/app-release.apk`
   (debug 版在 `apk/debug/`,两个都能直接装)

## 二、装进电视

任选一种:
- **U盘**:APK 拷进U盘 → 电视文件管理器打开安装
- **夸克网盘**:APK 传夸克 → 电视端下载安装(你的 TCL 已有夸克方案)
- **小米/TCL 电视助手 / ADB**:`adb connect 电视IP` → `adb install app-release.apk`

> 电视需开启「允许安装未知来源应用」(设置→系统/安全)。

## 三、手柄说明

- Xbox 手柄蓝牙配对**电视系统**(设置→遥控器与配件),不是配对游戏
- 游戏内原生拦截手柄按键,**不依赖 WebView 的 Gamepad API**(WebView 不支持该API,已在原生层解决)
- 键位:十字键/左摇杆移动,**A/X 开火**,**START 暂停/开始**,**B 退出游戏**
- 电视遥控器也能玩:方向键移动,OK键开火,菜单键暂停

## 四、常见问题

- **没声音**:已设置 `setMediaPlaybackRequiresUserGesture(false)`,如仍无声,按任意键后音频会激活(浏览器音频策略)
- **摇杆漂移误触发**:MainActivity.java 中 `DZ = 0.45f` 是死区,调大即可
- **想改键位**:改 MainActivity.java 的 `dispatchKeyEvent` 映射表
- **想改游戏**:游戏本体是 `app/src/main/assets/game.html`,纯前端,改完重新 Build 即可
